🛠️ Pengembangan Sistem
Aplikasi ini dikembangkan menggunakan arsitektur web dengan stack sebagai berikut:
- Frontend: HTML5, Bootstrap 5, JavaScript (Fetch API)
- Backend: Node.js, Express.js
- Database: MySQL (Laragon)
